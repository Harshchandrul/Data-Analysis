# Trek Operations Data Analysis

This project analyzes trek operations data to identify patterns in
profitability, occupancy, cancellations, and marketing performance.

## Objective
To support operational and business decisions for a trek-based travel startup
by analyzing historical trek data.

## Key Metrics Analyzed
- Revenue and profit by trek location
- Cancellations and risk by trek difficulty
- Occupancy trends across marketing channels
- Attendance trends over time
- Profit margin by difficulty level

## Key Insights
- Saan Valley and Mount Sumel treks generate the highest cumulative revenue
- Moderate difficulty treks offer the best balance of profit and attendance
- Instagram outperforms other channels in terms of occupancy
- Hard treks show higher cancellations with lower profit margins
- Post-monsoon months show higher attendance trends

## Tools Used
- Python
- pandas
- numpy
- matplotlib

## Dataset
Synthetic but realistic dataset modeled on real trek operations.
